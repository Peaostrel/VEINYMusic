# VEINYMusic // YANDEX RPC

![VEINYMusic](https://img.shields.io/badge/VEIN-Ecosystem-magenta?style=for-the-badge) ![Zero Config](https://img.shields.io/badge/Zero_Config-100%25-green?style=for-the-badge) ![Windows](https://img.shields.io/badge/OS-Windows_10%2B-blue?style=for-the-badge)

Бескомпромиссная интеграция Яндекс Музыки в Discord Rich Presence. 
Разработано для экосистемы **VEIN**. Никаких токенов, никаких костылей с расширениями браузера. Работает напрямую через ядро Windows.

---

## 🔥 ФИЧИ

- **System Tray Mode (Фоновый режим)**: Сверни окно консоли в трей и забудь. Управляй скриптом через иконку в области уведомлений.
- **Smart Auto-Update**: Скрипт автоматически проверяет наличие новых коммитов на GitHub при запуске и предлагает обновиться в один клик.
- **Zero-Install (Bootstrap)**: Больше не нужно вручную прописывать `pip install`. Скрипт сам проверит и доустановит все необходимые библиотеки при первом запуске.
- **Strict Mode (Строгий фильтр)**: Скрипт интеллектуально фильтрует звук в системе. Он отобразит в Discord *только* настоящую музыку. Если ты параллельно смотришь видео на YouTube или Twitch — VEINYMusic скроет это и сохранит твой профиль чистым.
- **Cloud Metadata Injection**: Скрипт на лету обращается к облачным серверам Яндекса, вытягивая высококачественные обложки альбомов и генерируя рабочие кнопки «Слушать» и «Альбом» для твоих друзей.
- **Анти-Лаг Система**: Идеальная синхронизация таймингов (Progress Bar) с браузером с точностью до миллисекунды.
- **Karaoke Status Sync (Слова в статус Discord)**: Скрипт в реальном времени транслирует текущую поющуюся строчку песни в ваш кастомный статус (Custom Status) в Discord. Идеально попадает в ритм, мгновенно сбрасывается при смене трека и автоматически возвращает ваш старый статус при паузе или выходе!
- **Tray Toggle & Hot Reload**: Полный контроль прямо из контекстного меню в трее. Включай/выключай трансляцию слов галочкой в один клик. Токен можно добавить без перезапуска — скрипт подхватит его на лету!

## 🎤 КАРАОКЕ В СТАТУСЕ DISCORD (СИНХРОНИЗАЦИЯ ТЕКСТОВ)

Вы можете транслировать слова проигрываемой песни прямо в свой **кастомный статус Discord** в реальном времени!

### ⚙️ Как настроить:

1. **Создайте файл**: Создайте пустой текстовый файл `discord_token.txt` в папке со скриптом (рядом с `yandex_presence.py`).
2. **Получите токен**:
   * Откройте Discord в браузере (Chrome / Yandex / Edge) и войдите в аккаунт.
   * Нажмите `F12` (или `Ctrl + Shift + I`) -> вкладка **Network** (Сеть).
   * Обновите страницу через `F5`.
   * В поле поиска (Filter) введите `/api`.
   * Выберите любой запрос (например, `library`), перейдите во вкладку **Headers** (Заголовки).
   * В разделе **Request Headers** найдите строку **`Authorization`**. Скопируйте длинный буквенно-цифровой код.
3. **Сохраните токен**: Вставьте скопированный токен в файл `discord_token.txt` и сохраните его.
4. **Запустите скрипт**: Программа сама увидит токен, напишет об этом в консоли и начнет трансляцию слов!

### 💎 Преимущества и тонкая настройка:

* **Интерактивный Трей**: Вы можете включать/выключать слова песен в один клик через пункт меню **«Слова песен в статусе Discord»** в трее. Если вы добавили токен при уже запущенной программе — просто нажмите этот пункт меню, и скрипт подхватит его на лету!
* **Умный бэкап**: Скрипт запоминает ваш исходный статус. Когда музыка ставится на паузу, выключается или программа закрывается — ваш оригинальный статус возвращается на место.
* **Тонкая регулировка таймингов**: Если слова спешат или отстают, откройте `yandex_presence.py` и измените значение `LYRICS_OFFSET = 0.8` (в секундах) в блоке конфигурации в начале файла.

## 🧠 КАК ЭТО РАБОТАЕТ (Архитектура)

Скрипт использует гибридную систему (Hybrid Sync), чтобы обойти необходимость авторизации по токенам:

1. **Захват аудио из ядра (SMTC):** Чтение официальной телеметрии ОС через `winsdk`.
2. **Точная математика времени:** Использование `last_updated_time` для синхронизации таймера с Дискордом.
3. **Облачная подкачка метаданных:** Запросы к API Яндекс Музыки для получения обложек 400x400 и метаданных альбомов.
4. **Автоматическое обновление:** Сверка локального SHA-коммита с актуальным через GitHub API.

## 🛠 ТЕХНИЧЕСКИЙ СТЕК

- **pypresence** — Асинхронная работа с Discord IPC.
- **pystray & Pillow** — Управление системным треем и иконкой.
- **winsdk** — Прямой доступ к Windows SMTC.
- **rich** — Отрисовка стильного TUI в консоли.
- **requests** — Анонимные запросы к API Яндекса и GitHub.

## ⚠️ ВАЖНОЕ ЗАМЕЧАНИЕ ДЛЯ FIREFOX

Из-за архитектуры **Mozilla Firefox**, этот браузер транслирует в систему только *одну* активную медиа-сессию. Если ты слушаешь Яндекс Музыку и параллельно запускаешь видео на YouTube в соседней вкладке — Firefox скроет музыку от Windows. 
* Решение: используй Яндекс.Браузер, Chrome, Edge или **Официальное приложение Яндекс Музыки для Windows** (рекомендуется).
