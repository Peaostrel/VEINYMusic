# VEINYMusic // YANDEX RPC

![VEINYMusic](https://img.shields.io/badge/VEIN-Ecosystem-magenta?style=for-the-badge) ![Zero Config](https://img.shields.io/badge/Zero_Config-100%25-green?style=for-the-badge) ![Windows](https://img.shields.io/badge/OS-Windows_10%2B-blue?style=for-the-badge)

Бескомпромиссная интеграция Яндекс Музыки в Discord Rich Presence. 
Разработано для экосистемы **VEIN**. Никаких токенов, никаких костылей с расширениями браузера. Работает напрямую через ядро Windows.

---

## 🔥 ФИЧИ (v2.5)

- **Zero-Config Архитектура**: Не нужно логиниться, искать токены или ставить плагины. Просто запусти и забудь.
- **Strict Mode (Строгий фильтр)**: Скрипт интеллектуально фильтрует звук в системе. Он отобразит в Discord *только* настоящую музыку. Если ты параллельно смотришь видео на YouTube или Twitch — VEINYMusic скроет это и сохранит твой профиль чистым.
- **Cloud Metadata Injection**: Скрипт на лету обращается к облачным серверам Яндекса, вытягивая высококачественные обложки альбомов и генерируя рабочие кнопки «Слушать» и «Альбом» для твоих друзей.
- **Анти-Лаг Система**: Идеальная синхронизация таймингов (Progress Bar) с браузером с точностью до миллисекунды за счет чтения внутренних метрик `last_updated_time` из Windows SDK.
- **Neon UI**: Стильная интерактивная консоль с живой полосой прогресса.

2. Скачай этот репозиторий.
3. Установи зависимости:

   ```bash

   ```

## 🧠 КАК ЭТО РАБОТАЕТ (Архитектура)

Скрипт использует гибридную систему (Hybrid Sync), чтобы обойти необходимость авторизации по токенам:

1. **Захват аудио из ядра (SMTC):** Скрипт сканирует аудио-сессии Windows через `winsdk`. Он не лезет в браузер и не крадет куки, а читает официальную телеметрию ОС.
2. **Точная математика времени:** Браузеры обновляют позицию трека в Windows с задержками. Скрипт берет метку времени последнего обновления (`last_updated_time`) и прибавляет к ней фактическое прошедшее время, добиваясь идеальной синхронизации таймера с Дискордом.
3. **Облачная подкачка метаданных:** Узнав название трека из ОС, скрипт незаметно делает запрос к открытому API Яндекс Музыки. Оттуда он мгновенно вытягивает 400x400 обложку, правильное название альбома и генерирует ссылки для кнопок, чтобы друзья могли слушать то же самое.
4. **Строгий фильтр:** Если Windows сообщает о видео с YouTube (или любом звуке, которого нет в базе Яндекса), скрипт включает защитный механизм и прячет статус, оставляя Discord чистым.

## 🛠 ТЕХНИЧЕСКИЙ СТЕК

- **pypresence** — Асинхронная работа с Discord IPC.
- **winsdk** — Прямой доступ к Windows Global System Media Transport Controls (SMTC) для перехвата аудио-потоков.
- **rich** — Отрисовка TUI (Terminal User Interface).
- **requests** — Легковесные анонимные запросы к `music.yandex.ru/handlers` для парсинга обложек.

## ⚠️ ВАЖНОЕ ЗАМЕЧАНИЕ ДЛЯ FIREFOX

Из-за архитектуры **Mozilla Firefox**, этот браузер транслирует в систему только *одну* активную медиа-сессию. Если ты слушаешь Яндекс Музыку и параллельно запускаешь видео на YouTube в соседней вкладке — Firefox скроет музыку от Windows. 
* Решение: используй Яндекс.Браузер, Chrome, Edge или **Официальное приложение Яндекс Музыки для Windows** (рекомендуется для идеальной работы).
